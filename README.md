# C语言模拟FAT文件系统｜操作系统课程设计
## 项目简介
本项目基于标准C语言实现简易FAT12文件系统模拟，在Windows+VSCode+GCC环境开发。
程序模拟物理磁盘，生成虚拟磁盘镜像 disk.img，完整复刻操作系统底层的磁盘管理、文件分配表、根目录管理机制，实现基础文件操作功能，完全满足操作系统课程设计要求。

## 核心实现功能
1. 虚拟磁盘初始化与格式化，自动生成 disk.img 磁盘镜像
2. BPB磁盘参数、FAT文件分配表、数据簇链式管理
3. 根目录项创建、查询、删除、覆盖逻辑
4. 四大核心功能：新建文件、新建目录、删除文件、文件复制
5. 磁盘簇占用状态可视化展示（空闲/已占用）
6. 目录文件信息可视化打印（文件名、大小、起始簇、文件属性）
7. 数据持久化，重启程序保留磁盘数据

## 运行环境
- 操作系统：Windows 10/11
- 编译工具：MinGW-w64 GCC
- 开发工具：VSCode

## 编译与运行教程
1. 进入项目根目录 FAT-OS-Code
2. 终端输入编译命令：
gcc src/main.c src/fat.c src/dir.c src/fileop.c -o fat.exe
3. 修复终端乱码后运行程序：
chcp 65001
.\fat.exe

## 项目文件结构
- src/          核心源码文件夹（7个功能代码文件）
- disk.img      自动生成的虚拟磁盘镜像文件
- fat.exe       编译生成的可运行程序
- README.md     项目说明文档
